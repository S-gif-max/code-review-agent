Automated Code Review Agent

Project Abstract

This project implements an Automated Code Review Agent using Generative AI. The system accepts source code in any language and produces structured, point-wise review feedback including issues, security risks, improvements, and a final verdict. The tool is designed for developer productivity and pull-request validation workflows.

Objectives

  • Automate pull-request style code review
  • Support any programming language
  • Provide structured one-line feedback
  • Ensure safe and stable AI output

Technologies Used

  • Python
  • Google Generative AI API
  • CLI Interface
  • Prompt Engineering

System Workflow

User Code → Prompt Builder → AI Model → Structured Review Output

Key Features

  • No language detection required
  • Works with any code format
  • Guaranteed structured output
  • Crash-safe response handling
  • API key masking for security

Conclusion

The Automated Code Review Agent reduces manual review effort and provides instant structured feedback using AI, making it suitable for academic projects and real development workflows.
